Evolution of Light

A Roguelike video-game written in Python using the curses library. The player must navigate through narrow dungeons, searching for more advanced technologies to use as a light-source that will fend off monsters.

The game is displayed onto a terminal in ASCII graphics. Noteable features are randomly generated dungeons and a field-of-vision algorithm. I am currently working on combat including light-source mechanics.
