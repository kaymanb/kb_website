Evolution of Light

A Rogue-like video-game written in Python using the curses library. Players will need to navigate a dungeon, fending off monsters as they delve deeper and deeper. The focus of the game is to progress through a timeline of lighting technology which will allow players to take control of the light that illuminates the dungeon levels. How the player chooses to light up the environment will determine whether they live or die.

The game is still under development. Most basic features of a rogue-like game are implemented, with a special focus on field of vision which will be used extensively as a game mechanic along with the lighting technologies. The next steps will be to smooth out NPC combat, as well as add some basic light mechanics.

You can download the source to demo the game.
