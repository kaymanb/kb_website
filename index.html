Evolution of Light

A Roguelike video-game written in Python using the curses library. Players will need to navigate a dungeon, fending off monsters as they delve deeper and deeper. The focus of the game is to progress through a timeline of lighting technology which will allow players to take manipluate the light that illuminates the dungeon levels. How to player chooses to light up the environment will determine if they live or die.

The game is still under development. Most basic features of a roguelike game are implemented, with a special focus on field of vision which will be used extensively as a game mechanic along with the lighting technologies. Next steps are to smooth out NPC combat, as well as some basic light mechanics.

You can download the source to play the game or see what it's like here.
